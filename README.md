# UEA Proyecto Tenis — Análisis de Novak Djokovic

Proyecto de análisis de datos de tenis enfocado en el rendimiento de Novak Djokovic, desarrollado para la Universidad de las Américas (UEA).

## Estructura del proyecto

```
DJOKOVIC/
├── notebooks/          # Jupyter notebooks de análisis
│   └── 01_exploracion_inicial.ipynb
├── docs/               # Documentación y recursos del proyecto
├── pbix/               # Archivos Power BI
├── data/               # Datos crudos (no versionados)
└── data_dictionary.txt # Diccionario de variables
```

## Datos

Los archivos de datos provienen del repositorio [Tennis Abstract Match Charting Project](https://github.com/JeffSackmann/tennis_MatchChartingProject) e incluyen estadísticas de partidos masculinos: direcciones de saque, profundidad de resto, puntos clave, puntos en la red, y más.

> La carpeta `data/` no está incluida en el repositorio. Descarga los CSV desde la fuente original.
